What it does

A small set of production-shaped n8n workflows you can import and use today — a daily RSS email digest, a webhook validation API, an uptime monitor, and a Google Sheets logger. Core nodes only, no community packages, no embedded secrets, and every one is validated import-clean with FlowProof (verdicts arbitrated against a live n8n import). Download the JSON, import it, fill in the obvious bits, activate.

Full install and usage in the README on GitHub.
